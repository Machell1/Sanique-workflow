----------------------------------------------------------------
  REQUIREMENTS
----------------------------------------------------------------

  1. Windows 10 or Windows 11
  2. Internet connection (first launch only - to download Electron)
  3. Node.js (the installer will guide you if it's not installed)

----------------------------------------------------------------
  INSTALLATION
----------------------------------------------------------------

  1. Extract this ZIP file to a folder on your computer.
     Recommended: C:\Program Files\CLAW\

  2. Double-click "run.bat"
     - On first launch, it will download Electron automatically.
       This takes 2-3 minutes depending on your internet speed.
     - On all subsequent launches, CLAW opens immediately.

  3. A desktop shortcut will be created automatically.

----------------------------------------------------------------
  FEATURES
----------------------------------------------------------------

  DASHBOARD        - Central command hub with truth monitoring
  SCHEDULE         - Court calendar with 3 terms, 4 rosters
  UPLOAD           - Multi-format document upload (PDF/DOCX/XLSX/PPTX)
  FILE CABINET     - Case folders, Record of Appeal assembly
  AGENT            - KIMI CLAW AI Legal Assistant
  VERIFICATION     - Judgment guidelines verification + citation check
  WORKFLOW         - 5-stage pipeline with bottleneck analysis
  GENERATOR        - Draft legal documents (advice, memos, judgments)
  AUDIT            - Immutable SHA-256 chain of custody
  SETTINGS         - User management, compliance, integrations

----------------------------------------------------------------
  AI TRUTH HARNESS
----------------------------------------------------------------

  100% VERIFIED     - No flag, full citation required
  99%  HIGH CONF    - Flagged for human research
  98%  ESCALATION   - Double-flagged, requires verification
  <98% BLOCKED      - Cannot be presented as fact
